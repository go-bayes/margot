# add or refresh a policy‑value test for one model

add or refresh a policy‑value test for one model

## Usage

``` r
margot_add_policy_p(model, depth = 2L, R = 999L, seed = 2025L)
```

## Arguments

- model:

  list. One element of \`margot\$results\`.

- R, seed:

  Passed to \[margot_compute_policy_value()\].
