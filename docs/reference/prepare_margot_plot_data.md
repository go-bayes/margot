# Prepare data for Margot plot

Prepare data for Margot plot

## Usage

``` r
prepare_margot_plot_data(
  .data,
  type,
  order,
  options,
  label_mapping,
  original_df
)
```

## Arguments

- .data:

  Input data frame

- type:

  Type of effect (RD or RR)

- order:

  Ordering method

- options:

  Options list

- label_mapping:

  Label mapping list

- original_df:

  Original data frame for back-transformation

## Value

Prepared data frame
