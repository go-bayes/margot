# Compute average conditional means within leaves

Compute average conditional means within leaves

## Usage

``` r
compute_leaf_means(
  leaf_idx,
  predictions,
  conditional_means,
  act_labels,
  leaf_names,
  use_math_notation = FALSE,
  output_format = "bullet",
  original_df = NULL,
  model_name = NULL,
  display_original_scale = TRUE
)
```
