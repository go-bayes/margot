# Generate extended academic-style report

Generate extended academic-style report

## Usage

``` r
generate_extended_report(
  evidence_summary,
  selection_results,
  rate_results_list,
  rate_results,
  qini_results,
  omnibus_results,
  models,
  spend_levels,
  label_mapping,
  alpha = 0.05,
  adjust = "BH",
  use_cross_validation = FALSE,
  cv_num_folds = NULL,
  cv_results = NULL
)
```
