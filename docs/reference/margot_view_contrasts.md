# Helper function to view flipped contrasts

Helper function to view flipped contrasts

## Usage

``` r
margot_view_contrasts(lmtp_output, table_type = "combined")
```

## Arguments

- lmtp_output:

  Output from margot_lmtp() or margot_resort_contrast_lmtp()

- table_type:

  Which tables to show: "combined", "individual", or "both"
