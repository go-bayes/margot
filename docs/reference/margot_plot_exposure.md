# Create a separate exposure plot

Create a separate exposure plot

## Usage

``` r
margot_plot_exposure(data, name_exposure, exposure_waves, baseline_wave)
```

## Arguments

- data:

  A data frame containing the exposure data

- name_exposure:

  The name of the exposure variable

- exposure_waves:

  A vector of wave numbers for exposure measurements

- baseline_wave:

  The wave number for baseline measurements

## Value

A ggplot object representing the exposure plot
