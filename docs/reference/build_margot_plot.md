# Build Margot plot

Build Margot plot

## Usage

``` r
build_margot_plot(
  .data,
  type,
  options,
  include_coefficients,
  standardize_label
)
```

## Arguments

- .data:

  Prepared data frame

- type:

  Type of effect (RD or RR)

- options:

  Options list

- include_coefficients:

  Whether to include coefficient text

- standardize_label:

  Type of standardization label (NZ, US, none)

## Value

ggplot object
