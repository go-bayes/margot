# Compute Average Gain for Simple Baseline

Compute Average Gain for Simple Baseline

## Usage

``` r
average_gain.qini_simple_baseline(object, spend, ...)
```

## Arguments

- object:

  A qini_simple_baseline object

- spend:

  The spend level (proportion treated)

- ...:

  Additional arguments (ignored)

## Value

A list with estimate and std.err
