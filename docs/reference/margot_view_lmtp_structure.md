# Helper function to view available models and contrasts

Helper function to view available models and contrasts

## Usage

``` r
margot_view_lmtp_structure(lmtp_output, show = "all")
```

## Arguments

- lmtp_output:

  Output from margot_lmtp() or margot_lmtp_resort_contrast()

- show:

  What to show: "models", "contrasts", "tables", or "all"
