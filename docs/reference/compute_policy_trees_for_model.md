# Compute policy trees for a single model

Compute policy trees for a single model

## Usage

``` r
compute_policy_trees_for_model(
  model_result,
  model_name,
  covariates,
  not_missing,
  custom_covariates,
  exclude_covariates,
  covariate_mode,
  compute_depth1,
  compute_depth2,
  train_proportion,
  verbose,
  seed
)
```
