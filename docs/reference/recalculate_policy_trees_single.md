# Recalculate policy trees for a single model

Recalculate policy trees for a single model

## Usage

``` r
recalculate_policy_trees_single(
  model_result,
  model_name,
  covariates,
  not_missing,
  custom_covariates,
  exclude_covariates,
  covariate_mode,
  verbose,
  seed,
  tree_method
)
```
