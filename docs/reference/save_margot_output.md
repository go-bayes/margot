# Save Margot plot output

Save Margot plot output

## Usage

``` r
save_margot_output(output, use_timestamp, base_filename, prefix, save_path)
```

## Arguments

- output:

  Complete output to save

- use_timestamp:

  Whether to include timestamp in filename

- base_filename:

  Base filename

- prefix:

  Optional filename prefix

- save_path:

  Directory to save in

## Value

No return value
